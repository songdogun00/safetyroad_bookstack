# SafetyRoad BookStack 사용자 가이드

이 문서는 **BookStack을 실제로 이용하는 최종 사용자용** 가이드입니다.
서버 설치/운영 관련 내용(Docker, 환경변수, 마이그레이션 등)은
[`README_install.md`](./README_install.md) 를 참고하세요.

## 1. 접속하기

- 주소: `http://<서버 주소>:8000` (실제 주소는 관리자에게 문의)
- 기본 관리자 계정: `admin@admin.com` / `password`
  - **최초 로그인 후 반드시 비밀번호를 변경**하세요. 
  (우측 상단 프로필 → `내 계정 설정` → `비밀번호 변경`)
- 개별 계정이 없다면 관리자에게 계정 생성을 요청하세요.

## 2. 기본 구조

BookStack의 문서는 아래 3단계 구조로 이루어집니다.

| 단위 | 설명 |
|------|------|
| **Book (책)** | 하나의 주제를 담는 최상위 단위. 예: "안전관리 매뉴얼" |
| **Chapter (챕터)** | Book 내부의 소제목 묶음 (생략 가능, 바로 Page를 넣어도 됨) |
| **Page (페이지)** | 실제 내용이 들어가는 문서 한 장 |

필요하면 여러 Book을 묶는 **Shelf(서가)** 도 만들 수 있습니다.

## 3. 문서 작성하기

1. 상단 메뉴 `Books` → `+ Create New Book` 으로 새 책 생성 (제목, 설명, 커버 이미지 선택 가능)
2. 책 안에서 `+ New Page` 또는 `+ New Chapter` 선택
3. 페이지 편집 화면에서:
   - **WYSIWYG 에디터**(기본) 또는 **Markdown 에디터** 중 선택 가능 (편집 화면 상단에서 전환)
   - 이미지·파일은 드래그 앤 드롭으로 바로 첨부 가능
   - 화면 우측 저장(`Save`) 버튼으로 저장, 이후에도 계속 수정 가능
4. 페이지/챕터/책 모두 우측 상단 `...` 메뉴에서 **이동(Move)**, **복사(Copy)**, **삭제(Delete)** 가능

## 4. 검색

- 상단 검색창에서 책/챕터/페이지 제목 및 본문 내용을 통합 검색할 수 있습니다.
- 태그를 걸어두면(`Page Tags`) 태그 기반으로도 검색/필터링이 가능합니다.

## 5. PDF 등으로 내보내기 (Export)

각 페이지·챕터·책 화면의 `...` 메뉴 → `Export` 에서 다음 형식으로 내보낼 수 있습니다.

- PDF
- Plain Text
- Markdown
- HTML

## 6. 권한/공유

- 관리자가 사용자별로 책/챕터/페이지 단위 열람·편집 권한을 설정할 수 있습니다.
- 특정 문서가 보이지 않거나 편집이 안 된다면 권한 문제일 수 있으니 관리자에게 요청하세요.

## 7. 자주 겪는 문제

| 증상 | 해결 방법 |
|------|-----------|
| 로그인이 안 됨 | 계정 존재 여부/비밀번호 재설정을 관리자에게 요청 |
| 페이지 저장이 안 됨 | 세션 만료 가능성 → 새로고침 후 재로그인, 내용은 임시저장(Draft)되어 있을 수 있음 |
| PDF에서 한글이 깨짐 | 관리자에게 문의 (서버 설정 문제, 사용자가 조치할 수 없음) |
| 이미지가 안 보이거나 깨짐 | 새로고침 후에도 반복되면 관리자에게 문의 |
